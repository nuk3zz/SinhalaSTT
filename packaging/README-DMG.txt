SinhalaSTT 0.1.0 beta

Install:
1. Drag SinhalaSTT.app to Applications.
2. Open SinhalaSTT from Applications or Spotlight.

Note:
This beta is unsigned and not notarized. If macOS blocks it the first time,
right-click SinhalaSTT.app and choose Open.

FFmpeg:
SinhalaSTT needs FFmpeg to read audio and video files. The app will not
install it automatically. If SinhalaSTT shows an FFmpeg setup note, install
FFmpeg once with Homebrew:

brew install ffmpeg

Output:
Generated SRT files save to Downloads.
Temporary WAV files save to Desktop/Cache/temporary wav.
