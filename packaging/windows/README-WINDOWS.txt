SinhalaSTT 1.0 — Windows

HOW TO RUN
1. Unzip this whole folder anywhere you like (for example your Desktop).
2. Open the folder and double-click SinhalaSTT.exe.

That's it. You do NOT need to install Python, FFmpeg, or anything else.
FFmpeg is already included inside this folder.

IMPORTANT
- Keep all the files together. Do not move SinhalaSTT.exe out of this folder
  on its own — it needs the files next to it to work.
- The first time you open it, Windows SmartScreen may show a blue
  "Windows protected your PC" box because this beta is not code-signed.
  Click "More info", then "Run anyway".

WHERE FILES ARE SAVED
- Finished subtitle (.srt) files are saved to your Downloads folder.
- Temporary audio files are stored in your system Temp folder and can be
  deleted at any time.

THE THREE TOOLS
- Text -> Subtitles: paste or open a script (PDF/DOCX/TXT), pick how to split it,
  and create an SRT. Sinhala text also gets an FM/DL legacy-font SRT automatically.
- Audio -> Subtitles (experimental): rough timing from audio; optionally fill in
  your script.
- AI Caption: optional. Uses your own Google Gemini API key over the internet.
  The Text and Audio tools work fully offline.
